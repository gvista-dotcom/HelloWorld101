Git Command Reference Table

List of common Git commands and their purpose
Command Description Category
Start a working area
clone Clone a repository into a new directory Start a working area
init Create an empty Git repository or reinitialize an existing one Start a working area
Work on the current change
add Add file contents to the index Work on the current change
mv Move or rename a file, a directory, or a symlink Work on the current change
restore Restore working tree files Work on the current change
rm Remove files from the working tree and from the index Work on the current change
Examine history and state
bisect Use binary search to find the commit that introduced a bug History & State
diff Show changes between commits or between commit and working tree History & State
grep Print lines matching a pattern History & State
log Show commit logs History & State
show Show various types of Git objects History & State
status Show the working tree status History & State
Grow & tweak your history
backfill Download missing objects in a partial clone History Management
branch List, create, or delete branches History Management
commit Record changes to the repository History Management
merge Join two or more development histories together History Management
rebase Reapply commits on top of another base tip History Management
reset Reset current HEAD to the specified state History Management
switch Switch branches History Management
tag Create, list, delete or verify a tag (signed with GPG) History Management
Collaborate
fetch Download objects and refs from another repository Collaboration
pull Fetch from and integrate with another repository or local branch Collaboration
push Update remote refs along with associated objects Collaboration
